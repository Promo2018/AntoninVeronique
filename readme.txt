	Rapide description : 
		Logiciel de gestion de clients, voyages et dossiers pour une agence de voyages.
		Le logiciel permet de rechercher, ajouter, modifier, supprimer des données d'une base de données qui contient :
		--- participants d'un voyage
		--- voyages et destinations
		--- assurances
		--- dossier contenant les 3 précédents.

	Prerequis :
		Installer Microsoft SQL Server Management Studio 2017 (moteur de base de données) sur l'ordinateur.
		Installer la base de données BoVoyage_VAV2 (nécessaire) en lançant fichier requête : "InstallationBoVoyage_VAV2.sql" et lancer tous les fichiers associés.
	
	Utilisation logiciel :
		Lancer Projet2_BoVoyage_VL-AP.exe.
		Lorsque que la console se lance, se connecter au compte Admin : nom de compte = Admin, mot de passe = admin.
		Logiciel composé en 3 menus : Dossier, Client, Voyages.
		--- dans chacun des menus, il est possible d'effectuer des actions associées aux menus
		--- pour effectuer ces actions, écrire les commandes affichées.
		--- pour quitter le logiciel, écrire la commande "quitter" dans la console.

		Ensuite, laissez-vous guider par le menu.
